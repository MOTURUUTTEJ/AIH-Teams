# Contributing to AIH-Teams

Thank you for contributing to AIH-Teams. Please follow this guide to keep collaboration smooth and review-friendly.

## Fork Instructions

1. Fork this repository to your GitHub account.
2. Clone your fork locally.
3. Add the original repository as `upstream` (recommended):
   ```bash
   git remote add upstream https://github.com/MOTURUUTTEJ/AIH-Teams.git
   ```
4. Create a new branch for your work:
   ```bash
   git checkout -b feature/<short-description>
   ```

## How to Submit Work

- Add your changes in a focused, clearly scoped branch.
- Include/update documentation when relevant.
- Commit with clear and descriptive commit messages.

## Code Standards

- Keep formatting consistent and readable.
- Prefer small, focused commits.
- Avoid committing secrets, credentials, or local artifacts.
- Ensure files and naming are clear and professional.

## Pull Request Submission Process

1. Push your branch to your fork.
2. Open a pull request against this repository.
3. Use a clear PR title and include:
   - What changed
   - Why it changed
   - Any validation steps performed
4. Address review feedback promptly.
5. Keep PR discussions respectful and collaborative.
